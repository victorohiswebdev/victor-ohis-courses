Pattern Matching

The biggest mistake beginners make when building bots is getting their Match Types wrong. Let's say your Menu tells the customer to "Reply 1 for Wigs." If you set the trigger for "1" as a Similarity Match, and a customer types "I want 10 wigs," the bot sees the "1" in "10" and fires the wrong script!

To prevent chaos, we use strict Pattern Matching.

• Use Similarity Match for greetings (e.g., "Hello" covers "Hello Chioma", "Hello there").
• Use Exact Match for menu numbers (e.g., "1" only fires if they type exactly "1" and nothing else).
