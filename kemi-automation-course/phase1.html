Building
The Digital Vault

Before we can build the pipes, we need a bucket. If our AI is going to extract valuable lead data from Mrs. Kemi's messy inbox, we need a clean, structured database to catch it automatically.

The Notion CRM

When high-ticket clients hear "CRM" (Customer Relationship Management), they immediately think of expensive software like Salesforce that costs thousands of dollars. We are going to build one for free using Notion.

An automation is basically a digital delivery guy. For the delivery guy to know where to put the package, the house needs clear room labels. In our Notion database, we must create exact "Properties" (columns) so the AI knows exactly where to drop the Client's Name, their Budget, and the Meeting Summary.

The Execution

Setting up the Database Columns.

The Database Schema

"Open a free Notion account and create a new 'Table Database'. We are going to add 4 specific columns that our AI will look for:"

  • 1. Client Name (Title property)
  • 2. Email Address (Email property)
  • 3. Investment Budget (Number property)
  • 4. AI Extracted Summary (Text property)
01. Create Workspace

Log into Notion and create a blank page. Type `/table` and select "Table Database - Inline".

02. Add Properties

Rename the columns exactly as shown above. Spelling matters, because our automation tool will look for these exact names.

03. Leave it Empty

Do not type any data into it! The entire point of this masterclass is that the robot will fill this table for us.
