Injecting
The Intelligence

We have the Gmail trigger waiting for an email. Now, we need to plug the AI into the pipeline to read that email. This is where we stop talking to AI like a human, and start programming it like a machine.

Structured Prompting

When you use ChatGPT or Gemini normally, it is polite. If you ask it to extract a name, it says, "Sure, Victor! The name is John."

In an automation, if the AI says "Sure, Victor!", it will ruin our Notion database because Notion is expecting raw data, not a conversation. We must use Structured Prompting. We are going to strictly command the AI to strip away all politeness and return only the raw data (Name, Budget, Summary) separated by commas, so our database can read it cleanly.

The Execution

Connecting the Google Gemini API Module.

The Machine Prompt

"Read the following email text. Extract the Client Name, Investment Budget, and write a 1-sentence Summary.

CRITICAL RULE: Do not use conversational text. Return the data exactly in this format:
Name: [Name]
Budget: [Budget]
Summary: [Summary]"

01. Add AI Module

In Make.com, click the plus sign next to the Gmail bubble. Search for "Google Gemini" and select "Generate Text".

02. Map the Data

Paste the Machine Prompt into the text box. Then, click and drag the "Email Content" variable from the Gmail bubble right below it.

03. The API Key

You will grab your free API key from Google AI Studio and paste it in to authenticate the connection.
